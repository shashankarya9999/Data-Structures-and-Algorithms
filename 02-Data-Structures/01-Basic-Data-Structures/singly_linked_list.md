2) Operations on Singly-linked list and their Time-Complexities:

| Singly-Linked List | No Tail | With Tail |
|:-------------------|:-------:|:---------:|
| PushFront(Key) | O(1) |
| TopFront() | O(1) |
| PopFront() | O(1) |
| PushBack(Key) | O(n) | O(1) |
| TopBack() | O(n) | O(1) |
| PopBack() | O(n) |
| Find(Key) | O(n) |
| Erase(Key) | O(n) |
| Empty() | O(1) |
| AddBefore(Node,Key) | O(n) |
| AddAfter(Node,Key) | O(1) |
